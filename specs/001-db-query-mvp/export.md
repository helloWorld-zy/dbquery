# 结果导出说明

本文档描述前端结果导出能力与后端导出接口的使用方式。

## 功能概览

- 支持将查询结果导出为 CSV、JSON。
- 导出控件位于“查询结果”区域，支持选择格式后导出。
- 当没有可用结果时，导出按钮不可用并提示原因。

## 前端使用

1. 执行查询获得结果。
2. 选择导出格式（CSV 或 JSON）。
3. 点击“导出”按钮。
4. 成功后会提示下载地址。

## 接口说明

- `POST /api/v1/exports`

请求体示例：

```json
{
  "queryId": "<query_request_id>",
  "format": "csv"
}
```

响应示例：

```json
{
  "downloadUrl": "http://.../export/<file>.csv"
}
```

## 注意事项

- 仅支持导出已执行查询的结果。
- 导出文件用于临时下载，建议及时保存。
- 后续可扩展其他格式（如 XLSX、Parquet）。
